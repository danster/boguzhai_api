###<a name="1">历史竞拍</a>
Http Request: 

```
m=getHistoryAuctionList
sessionid=1h283019400031n22hhu22313145252 
number= #分页序号
```
Http Response:

- code=0 : 获取成功
- code=-1 : sessionid 错误
- code=1 : 获取失败

```
{ 
    "code": 0;
    "data": {
    	"count": "35"; #结果总数
    	"size": "8";   #每次分页数目
    	"list":[ 
	    	{
                "id": "";   #拍品ID
                "no": "";   #拍品图录号
                "name": ""; #拍品名称
                "image": "http://..../29102.png";
                "startPrice": "1000"; #起拍价
                "dealPrice": "11000"; #成交价
                "biddingCount": "18"; #出价次数
                "myTopPrice": "2400.00" #我的最高出价
	    	},
	    	...
	     ]
	   }
}
```
