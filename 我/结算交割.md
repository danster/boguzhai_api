###<a name="4">提交订单信息</a>
Http Request: 

```
m=submitOrderInfo
sessionid=1h283019400031n22hhu22313145252 
auctionId=1232142,341351 #拍品ID之间用逗号隔开
addressId=  #收货地址id
deliveryType= #配送方式 1自提 2快递
deliveryPersonName=    #提货人姓名(选择自提方式)
deliveryPersonMobile=  #提货人手机(选择自提方式)
deliveryPersonType=    #提货人有效证件类型(选择自提方式):"二代身份证" "三代身份证" "港澳台身份证" "护照" "其它"
deliveryPersonNumber=  #提货人有效证件号码(选择自提方式)
checkCode=             #提货验证码(选择自提方式)
expressSupport=        #是否保价(选择快递方式)： 1保价 0不保价
supportPrice=          #保费(选择快递方式)
invoiceHeaderType=     #发票抬头类型："个人" "公司"
invoiceHeader=         #发票抬头
invoiceContent=        #发票内容 "文法四宝" "古籍书刊" ....
remark=                #订单备注即买家留言
```
Http Response:

- code=0  : 提交信息成功，返回订单ID
- code=-1 : sessionid 错误
- code=1  : 提交信息成功
``` 
{ 
    "code": 0;
    "data":{
    	"orderId": ""; #订单编号
    }
}
```

###<a name="7_2">订单支付</a>
Http Request: 

```
m=payOrderById
sessionid=1h283019400031n22hhu22313145252 
orderId= #订单ID 
useBalance= # 是否使用暂存款 1使用 0不使用
useDeposit= # 是否使用保证金 1使用 0不使用
type= #网银支付方式 0不使用任何网银支付 1使用银联支付 
```
Http Response:
> type=0时,不使用任何网银支付,使用暂存款或保证金或两者都用

- code=0  : 支付成功
- code=-1 : sessionid 错误
- code=1  : 金额不足,支付失败
- code=2  : 其它原因,支付失败
``` 
{ 
    "code": 0;
}
```

> type=1时,使用银联支付方式支付，暂存款，保证金也可以使用

- code=0  : 返回支付链接成功
- code=-1 : sessionid 错误
- code=1  : 暂存款和保证金已足够支付，且支付成功
- code=2  : 支付失败
``` 
{ 
    "code": 0;
    "data": {
    	"payUrl": "";  #银联支付网络支付链接
	}
}
```
