### <a name="1">获取拍卖会列表接口</a>
Http Request: 

```
按条件查询到的拍卖会列表
m=getAuctionMainList
type= #拍卖会拍卖方式: "" "同步" "网络"
status= #拍卖会状态: "" "预展中" "拍卖中" "已结束"
```
```
获取首页的拍卖会列表
m=getHomeAuctionMainList
```

Http Response:

- code=0 : 获取信息成功，返回拍卖会ID列表
- code=1 : 获取信息失败，服务器错误
